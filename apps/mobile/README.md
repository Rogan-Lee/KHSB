# 스터디룸 매니저 모바일

Expo SDK 56과 Expo Router로 구성한 iOS 우선 모바일 앱입니다. 동일 코드베이스로 Android 출시를 준비합니다.

## 현재 범위

- 학생/관리자 역할 선택과 역할별 라우트 가드
- SecureStore 기반 로컬 세션 경계
- 학생 홈, 관리 프로그램, 질의응답, 내 정보
- 관리자 홈, 입퇴실, 멘토링, 질의응답, 근무·운영 메뉴
- 운영 API 연결 전 개발용 데모 세션

## 실행

정상 동작하는 Node.js 22 이상 환경에서 실행합니다.

```bash
npm install
npm run ios
```

웹 레이아웃 확인:

```bash
npm run web
```
