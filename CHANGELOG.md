# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1.0] - 2026-03-27

### Added
- 입퇴실 일정 탭에서 외출/복귀 시간 직접 설정 가능 (요일별 다중 외출 일정, 사유 포함)
- 상벌점 내역 테이블 정렬 기능 (날짜/이름/점수 오름차순·내림차순)
- 기간별 상벌점 조회 정렬 기능 (순점수/상점/벌점/이름/건수)
- 주간 멘토링 계획: 일자별 추가 버튼에서 검색으로 담당 외 학생도 배정 가능
- AI 멘토링 리포트 작성 기능 (Groq LLaMA 3.3 70B)
- 회의록 기능 (팀별 탭 지원)
- 인수인계/투두/루틴 통합 관리
- 카카오 메시지 UI 개편 + 공부계획 리포트 페이지
- 메시지 발송 템플릿/학생 선택 검색 콤보박스
- 상벌점 월별 랭킹/기간별 조회
- 구글 시트 연동 (원생 정보, 성적 CSV 가져오기)
- 멘토링 학생정보 블러 처리

### Changed
- 입퇴실/외출 일정 저장을 Prisma 트랜잭션으로 원자적 처리
- 주간 멘토링 계획에 담당 외 배정 학생도 표시
- KST 기준 시간 통일
- 출결표 가로 스크롤 개선 (Shift+휠, 좌우 버튼), 인라인 편집 개선

### Fixed
- MeritRecord 타입명 충돌(Record 내장 타입) 수정 → 빌드 오류 해결
- middleware.ts → proxy.ts 마이그레이션 (Next.js 16 deprecated)
- Vercel 배포 리전 sin1(싱가포르) → icn1(서울) 변경
- 구글 캘린더 이벤트 전체 범위 조회 및 동적 로딩 개선
- 입실 기록 없을 때 결석 표시, 실제 외출 기록 없으면 외출중 미표시
- 주간 멘토링 계획 낙관적 상태 실패 시 DB 동기화 롤백
- AI 리포트 max_tokens 조정 및 프롬프트 개선
