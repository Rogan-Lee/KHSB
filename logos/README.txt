대학 로고 마퀴 — 로고 이미지 넣는 곳
=====================================

landing.html 의 "재원생 합격·재학 대학" 마퀴와 후기 카드의 대학 로고는
이 폴더의 PNG 를 표시합니다. 파일이 없으면 대학 이름 텍스트 칩으로 자동 표시됩니다.

붙여주신 로고 → 저장할 파일명 (landing.js 의 UNIS 목록 기준):
  연세대        →  yonsei.png
  서울대        →  snu.png
  고려대        →  korea.png
  성균관대      →  skku.png
  한양대        →  hanyang.png
  서강대        →  sogang.png
  이화여대      →  ewha.png
  중앙대        →  cau.png
  경희대        →  khu.png
  한국외대      →  hufs.png
  서울시립대    →  uos.png

권장 사양:
  - 가로형 워드마크(가로로 긴 형태) 권장, 높이 약 80~120px, 배경 투명 PNG
  - 마퀴/후기 카드에서 흑백으로 표시되고, 마우스를 올리면 원래 색으로 보입니다.
  - SVG 로 쓰려면 landing.js 의 확장자(.png)도 .svg 로 바꾸세요.

주의:
  - 대학 엠블럼/로고는 각 대학의 상표입니다. 실제 사용 권한이 있는 로고만 넣어주세요.
  - 대학을 추가/변경하려면 landing.js 의 UNIS 배열을 수정하세요.
