## 실행 예정 쿼리 (DB 접근 가능 시)

### 1. 대기/진행 중 항목 조회

```sql
SELECT
  id,
  title,
  category,
  priority,
  requester,
  "authorName",
  "createdAt"::date AS created,
  status
FROM "FeatureRequest"
WHERE status IN ('PENDING', 'IN_PROGRESS')
ORDER BY
  CASE priority
    WHEN 'URGENT' THEN 0
    WHEN 'HIGH'   THEN 1
    WHEN 'NORMAL' THEN 2
    WHEN 'LOW'    THEN 3
  END,
  "createdAt" ASC;
```

### 2. 최근 24시간 댓글 조회

```sql
SELECT
  frc.content,
  frc."authorName",
  fr.title,
  frc."createdAt"
FROM "FeatureRequestComment" frc
JOIN "FeatureRequest" fr ON fr.id = frc."featureRequestId"
WHERE frc."createdAt" > NOW() - INTERVAL '24 hours'
ORDER BY frc."createdAt" DESC;
```
